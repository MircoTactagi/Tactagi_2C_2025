Ejercicio 4, 5 y 6 - Control de Display BCD Multiplexado
Descripción del Proyecto
Este proyecto implementa un sistema de control para un display de 7 segmentos multiplexado con codificación BCD, utilizando GPIOs de un microcontrolador ESP32. El sistema permite visualizar números de hasta 3 dígitos mediante la técnica de multiplexación temporal.

Características Principales
Conversión a BCD: Conversión de números enteros a formato BCD (Binary-Coded Decimal)

Control GPIO: Manejo de pines GPIO para controlar displays de 7 segmentos

Multiplexación: Implementación de multiplexación temporal para displays de múltiples dígitos

Soporte para 3 dígitos: Capacidad de mostrar números desde 0 hasta 999
